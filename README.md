# TGED Farmbot
An open-source farmbot written in Python for *The Greatest Estate Dev: Squad TD*.

[Discord](https://discord.gg/fAyes2ErZV)

## Setup
**This tutorial currently should support both android & iOS devices.**

1. Download this repository as a `.zip` file and extract it.  
2. Download **Proxyman** on your iOS device and set it up.  
3. Open the game, log out, then log back in.  
4. Open Proxyman, find `tino.detailgames.ai`, and enable **SSL Proxying**.  
5. Close and reopen the game, then wait until you reach the main screen.  
6. Open Proxyman again and find `https://tino.detailgames.ai/auth/refresh`.  
7. Go to **Response â†’ View Body**, click the three dots in the top-right corner, and select **Copy**.  
8. Send the copied message to your computer (via Discord or email).  
9. Paste the text into `tokens.txt` and launch the farmbot.  

**Warning:** Do **NOT** open the game while using the farmbot!

(Sorry for the complicated setup, I don't really know any other way to make this work.)
